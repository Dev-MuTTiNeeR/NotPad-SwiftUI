# NotPad 📝

A minimalist, fast, and user-friendly iOS note-taking application. Built with a strict adherence to Apple's native design language, prioritizing a seamless user experience.

## ✨ Features

* **Dynamic Interface:** Standard layout for short titles, with an auto-expanding smart title area for longer text.
* **Auto-Management:** Empty notes are automatically discarded when exiting the detail view to keep the list clean.
* **Instant Save:** Every change made to your notes is instantly and synchronously saved to the local database.
* **Native Experience:** Full Dark Mode compatibility and fluid transitions that follow iOS standards.

## 🛠 Technologies Used

* **Swift & SwiftUI:** Modern, declarative, and reactive user interface design.
* **Core Data:** Secure, fast, and offline on-device data storage.
* **MVVM Architecture:** Efficient state and data management utilizing `@Environment` and `@ObservedObject`.
